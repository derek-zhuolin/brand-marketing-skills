# 品牌营销 AI Skill 工具箱

精选品牌营销相关的 AI Skill（GitHub 高星 / 多人验证），按「调研 → 策略 → 写作 → 视觉 → 自动化」分类，并给出可直接复制的 prompt 和工作流。

- 在线访问：https://derek-zhuolin.github.io/brand-marketing-skills/
- 技术：单页静态站，零依赖。`index.html` = 页面与逻辑，`skills-data.js` = 全部 skill 数据（唯一内容来源）。
- 本地预览：直接双击 `index.html` 即可。

## 想加 / 改 skill？

两个人协作，请先读 [CONTRIBUTING.md](./CONTRIBUTING.md)——所有内容只改 `skills-data.js`，走分支 + PR 互审后合并到 `main`。`main` 永远是发布版本。
