自己组一个

不需要懂技术 4 步就能搞定

01
写下你想让 AI 帮你做的事
举例 "这周帮我把一个品牌活动做成落地页和配套内容"
02
拆成几个步骤 → 每步找对应的 Skill
① 研究目标用户和竞品页面 → 🔍 调研洞察
② 确定定位和页面结构 → 🧠 定位策略
③ 写落地页和社媒文案 → ✍️ 内容写作
④ 去 AI 味 → ✍️ 写内容（humanizer-zh）
⑤ 正文配图 / 封面 / 案例图 → 🎨 内容视觉化
⑥ 落地页转化和飞书沉淀 → 🤖 页面转化
03
在对应分类里挑 Skill
挑出来 competitor-profiling · positioning-messaging · copywriting · humanizer-zh · baoyu-cover-image · page-cro
从「Skill 列表」点开看安装命令 全部复制粘贴装好
04
粘进 AI 一句话让它全部跑完
请按以下步骤帮我做"品牌活动落地页 + 配套内容"： 1. 用 competitor-profiling 拆 3 个同类活动页面的定位、结构和 CTA 2. 用 positioning-messaging 写一句话定位、目标用户和核心利益点 3. 用 copywriting 写落地页首屏、模块文案和报名 CTA 4. 用 humanizer-zh 全部过一遍，去掉生硬 AI 味 5. 用 baoyu-cover-image 或 ian-xiaohei-illustrations 给我 3 个视觉方向 6. 用 page-cro 检查页面转化路径，并列出需要回收的内容资产 请一口气跑完 不要每步等我
不需要懂原理 装上就能用
新手入门 · 10 分钟

装 AI Agent → 复制 → 粘贴 → 用

不需要会写代码 装一个 AI Agent 从这个页面复制安装命令粘进去 AI 会自己装好

01
装一个 AI Agent

国内外都行 挑一个对你最方便的——所有这些工具都能加载本页推荐的开源 skill 如果你不知道选哪个 从 Claude Code 开始

npm install -g @openai/codex

WorkBuddy 国内
用国内的 AI Agent？ 不用学命令行——把链接粘给它 它自己会装：
帮我把这个仓库装好 让我能用里面的所有 skill： https://github.com/coreyhaines31/marketingskills https://github.com/jinggreen15/ai-design-team https://github.com/op7418/Humanizer-zh
02
在本页找你要的 skill 点「复制」

本页上下所有代码块右上角都有「复制」按钮 点一下命令进剪贴板 skill 卡点开后里面的安装命令、prompt 模板全都能一键复制

# 比如装 marketingskills（39 个营销 skill 一锅端） git clone https://github.com/coreyhaines31/marketingskills \ ~/.claude/skills/marketingskills
→
已进剪贴板
03
粘进 AI Agent 再喊具体的小 skill

把命令粘进 AI Agent 它会自己 git clone、把 skill 装到正确位置 但要注意：装完仓库不等于自动会用 你得明确喊出要用哪个小 skill——AI 才会去查那个 skill 的 SKILL.md 按指引干活

# 装完后 第一句这样喊 — 明确点名要用的 skill 用 brief-analyst 拆这份业配需求 再用 content-planner 给我 3 个切角 最后用 humanizer-zh 过一遍去 AI 味输出
💡 看到 AI 按 "brief-analyst → content-planner → humanizer-zh" 顺序切换工作风格 就说明它正确加载了 skill 不喊名字 AI 只会用通用模式回答 那 skill 就白装了
补充说明 · Skill 之外还需要什么

Skill 负责思路和流程
生图 / 生视频 / 发布要靠外部工具

Skill 教 AI 怎么想——比如 "用 image 写一个生图 prompt" 但要真出图 需要接 ChatGPT Image 2.0 这种外部模型 要写入飞书、Notion 或 Google Workspace 需要接对应 外部连接

装完 skill 之后你还要做两件事： ① 给 AI Agent 配上你常用的外部模型 API key ② 配好对应的外部连接（飞书 / Notion / Google 等）

生图模型
ChatGPT Image 2.0 · Midjourney · Sora-Image · 即梦
生视频模型
Seedance 2.0 · Sora · Runway · 即梦视频
外部数据 / MCP
飞书 CLI · Notion API · Google APIs
完整配置 AI Agent（豆包/Claude Code/...） + Skill 包（这页 30+ 个 Skill） + 外部模型（生图生视频） + 外部连接（飞书/Notion/Google） = 真正能跑的自动化工作流
